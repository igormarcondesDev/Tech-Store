# 🛍️ Tech Store

Uma **landing page moderna de e-commerce de tecnologia** desenvolvida com **HTML, CSS e JavaScript puro**, apresentando um design interativo com slider de produtos, animações suaves e layout responsivo.

O projeto simula a interface de uma **loja virtual de fones de ouvido**, destacando produtos com mudanças dinâmicas de cor, imagem e fundo conforme a navegação do usuário.

🔗 **Demo do projeto**  
https://igormarcondesdev.github.io/Tech-Store/

---

# 🚀 Funcionalidades

- 🛒 Interface inspirada em **e-commerce moderno**
- 🎧 **Slider interativo de produtos**
- 🎨 **Mudança dinâmica de cores e background**
- ✨ **Animações suaves com CSS**
- 📱 **Design responsivo** (desktop, tablet e mobile)
- ⚡ Transições visuais entre produtos
- 🧭 Navbar simples e minimalista
- 🖼️ Destaque visual para produto principal

---

# 🧱 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias fundamentais do **Front-End**:

- **HTML5** → Estrutura da página
- **CSS3** → Estilização, layout e animações
- **JavaScript (Vanilla JS)** → Interatividade e controle do slider
